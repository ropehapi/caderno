# Como fazer frontends melhores
Quer fazer seu frontend sair do básico para algo realmente bonito em uma linha? É bem simples.

## Os comandos
Dentro do Claude Code, rode:
```sh
/plugin marketplace add anthropics/claude-code
```

Depois instale a skill de frontend:
```sh
/plugin install frontend-design@claude-code-plugins
```

## O que isso faz
Você está baixando o personal spec da Anthropic (a empresa que fez o Claude Code) e instalando a skill de frontend design.

Com isso, você vai ensinar o seu Claude Code — que antes era meio cru ao utilizar frontend — a usar a skill de frontend de uma forma muito mais eficiente.

## Resultado
O Claude Code vai passar a gerar frontends muito mais bonitos e profissionais, seguindo as melhores práticas de design.

Se você quiser ver mais exemplos, é só tentar que você vai ver que vai dar muito certo.
