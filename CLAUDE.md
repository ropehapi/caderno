# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repository is

A personal digital study notebook written in Portuguese. All content is Markdown — there is no build system, no tests, and no executable code. The purpose is to document theoretical knowledge from courses and books, serving as a reference for the community.

## Structure

```
Arquitetura/        # Software architecture notes (APIs, design patterns, microservices, testing)
Desenvolvimento/    # Language-specific notes (Go, Elixir, Javascript)
Devops/             # Infrastructure notes (AWS, Docker, Kubernetes, GCP, SQL, CI, git)
IA/                 # AI tooling setup guides (Claude Code, MCP servers)
OLD/                # Legacy notes pending migration to the current structure
src/                # Supporting assets (images, book PDFs, certificates)
```

## Conventions

- All notes are written in **Portuguese (pt-BR)**.
- Files are numbered with a prefix (`1_`, `2_`, `010_`) to indicate reading order within a topic.
- Each topic lives in its own subdirectory; new subjects get their own folder following the existing naming pattern.
- The `OLD/` directory mirrors the current structure but holds older, unreviewed content that is being progressively migrated.
