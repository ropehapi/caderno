# Extension manager configs - Ubuntu
Abaixo, segue um passo a passo para a instalação e configuração das extensões que eu uso no meu ubuntu:

1. Instalar extension-manager
``` bash
sudo apt install gnome-shell-extensions gnome-shell-extension-manager chrome-gnome-shell
```
2. Executar o gerenciador
```
extension-manager
```
3. Instalar as seguintes extensões:
- Vitals
- Blur my shell
- Clipboard-indicator
- Rounded window corners reborn
